# Top Screen Door Dashboard

Standalone Kalodata creator-content dashboard for magnetic screen-door products.

- This repository is intentionally isolated from `creator-briefs` and `top-bidet-dashboard`.
- `index.html` is the stable latest edition.
- Daily archives are stored under `screen-door-brief/YYYY/MMDD.html`.
- Data is refreshed from the authenticated Kalodata browser session and published through GitHub Pages.
